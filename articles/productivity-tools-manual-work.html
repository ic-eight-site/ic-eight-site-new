A Lot of Productivity Tools Still Assume a Human Is Doing the Manual Work

I quit Notion. Not because it's bad — because the workflow it was built for isn't mine anymore. · Published 2026

I started using Notion because GPT recommended it.

The idea was to collect ideas there and turn them into articles. I'd been posting sporadically from my personal account since July, and impressions were healthy. Then in November I switched to posting twice a week, and impressions crashed to single digits almost immediately. Grok's advice was blunt: post volume builds algorithmic trust over time. So I started using the same Notion list to count — a way to make sure I was actually posting enough, and to feel like the slow grind was leading somewhere.

Notion seemed like the obvious tool either way. Everyone uses it. I took the tutorial. I still didn't get it.

What bothered me wasn't really the lack of formatting. It was something more specific: paragraph spacing collapsed every time I pasted text in. And the block-based editing style never clicked for me. I like HTML. I don't like Notion's markup.

For a while I assumed this was a "me" problem — a tool everyone finds useful, and I just hadn't learned it yet.

But today I looked at why I actually kept using it, and the reasons no longer exist.

I don't need to track post count anymore. I've posted enough by now that the data already told me what I needed to know: volume alone doesn't move the algorithm in any way I can predict. Some posts get 9 impressions, some get 47,000. Counting stopped being useful months ago.

The other reason I liked Notion — quick access to past drafts — turned out to be solved differently too. I don't write alone anymore. When I want to revise something, I paste it into Claude and say "fix this paragraph, keep the rest." I'm not formatting, tagging, or organizing anything by hand. I'm deciding. The editing — the actual hands-on-keyboard part — happens somewhere else now.

That's when it hit me: Notion is built for a workflow where the human does the structuring. Tagging, filtering, organizing, formatting — by hand. I don't do that part anymore. I think, I direct, and the execution happens through AI. The tool I needed was never a database. It was just something that wouldn't mangle my paragraph breaks. Apple Notes does that fine.

This isn't a Notion review. It's a small example of something larger: a lot of "productivity" tools assume a human is still the one doing the manual work. Once that assumption breaks, a surprising number of beloved tools quietly stop being useful — not because they got worse, but because the job they were built for isn't yours anymore.

This is the same shift IC Eight is built around. We're not asking AI to translate what a human already decided to write. We decide what should be generated, AI generates it, and a human reviews the result for intent, not correspondence. The structuring work moved. The judgment didn't.

Notion didn't get worse. My workflow just stopped needing what it does.
